h3. 10 de enero de 2010

Agregue muchas cosas para trabajar con javascript: El env.rhino.js para poder usar
el objeto window en la implementacion de javascript en java, Rhino.

h4. Como se usa

* Abrir la consola de javascript, Rhino
  M-x run-js

* Cargar el objeto window, con env.rhino.js (mas informacion google envjs)
  M-x wally-load-envjs

* Cargar $W().values y $W().methods que hice
  M-x wally-load-jsreflector

* Ahora cargar algun HTMLDocument
  js> window.location = "http://news.ycombinator.com"
  o tambien puede ser 
  js> window.location = "http://localhost:3000/"

* Ahora que ya se tiene un document, se puede cargar una libreria como jQuery, jQuery con jQuery tools, Prototype, o rightjs
  M-x wally-load-jquery
  M-x wally-load-jquery-tools      ;; ya tiene incluido jquery
  M-x wally-load-prototype
  Se puede cargar cualquier libreria de Javascript escribiendo load("libreria.js")

h1. Acerca de $W()

*$W()* es un wrapper que hice dentro del cual se puede poner cualquier objeto
de javascript y llamar $W(objeto).attributes, $W(objeto).values, $Wobjeto().methods, $W(objeto).print_methods()
para saber mas acerca de lo que se puede hacer con ese objeto.
